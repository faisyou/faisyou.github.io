WhatsApp Campaign Management

B2B Portal • Campaign Provider • Genesys Cloud

Overview

How B2B Customers run WhatsApp campaigns while the Campaign Provider and Genesys Cloud manage execution.

Parties Involved

  • B2B Customer – initiates campaigns.
  • Campaign Provider – owns customer data and manages execution.
  • Genesys Cloud – handles calling lists and message delivery.

Key Principles

  • Only pre-approved WhatsApp templates can be used.
  • B2B Customers do not see phone numbers.
  • Provider maintains a per-B2B Do Not Contact (DNC) list.

B2B Portal Journey

Steps performed directly by the B2B Customer.

Portal Steps

  • Login to the Campaign Provider Portal
  • Create or select a campaign
  • Select a pre-approved WhatsApp template
  • Apply filters (segment, demographics, etc.)
  • Launch the campaign

Examples of Filters

  • Location
  • Age group
  • Gender
  • Customer segment

Behind the Scenes: Genesys Cloud

What happens after the B2B Customer launches the campaign.

Creating Campaign & Calling List

  • Provider backend receives launch request
  • Filters are applied and DNC customers removed
  • Genesys Outbound Campaign is created via API
  • Calling list created and populated

Execution & Visibility

  • Provider launches campaign in Genesys Cloud
  • B2B Customer sees only aggregate statistics
  • Actual WhatsApp sending is done via Genesys integration

Customer WhatsApp Experience

End-customer receives a templated message with buttons.

Buttons in the Template

  • Interested – expresses interest
  • Not Interested – declines
  • Opt Out – added to DNC

Callback Handling

  • WhatsApp → Genesys callback
  • Genesys → Provider API call

DNC & Responsibilities

How opt-out is enforced and who owns what.

DNC Handling

  • Opt-out adds customer to B2B-specific DNC.
  • DNC is applied before every campaign.
  • Customer only stops messages for that specific B2B brand.

Responsibilities

  • B2B Customer: Configures and launches campaigns
  • Campaign Provider: Manages data, execution, DNC
  • Genesys Cloud: Sends WhatsApp messages and triggers callbacks
